oneshotGlowHacks

C++
C++

Stary (legacy) glowhack i triggerbot do CSGO (zewnętrzny).

Stary projekt uczący podstaw C++, zarządzania pamięcią procesów Windows i niskopoziomowych operacji.

Dlaczego?

Aby poznać API Windows oraz wzorce skanowania pamięci w praktyce przy czymś interesującym.

Co zawiera?

  • Dostęp w czasie rzeczywistym do pamięci procesu CSGO
  • Skanowanie wzorców pamięci
  • Wielowątkowe operacje pamięci
  • ESP z kolorami / filtrowanie drużyn
  • Triggerbot z opóźnieniami zależnymi od broni
