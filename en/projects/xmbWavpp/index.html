xmbWavpp

C++

Command-line animation that replicates the iconic wave effect from the PlayStation 3 XMB (Cross Media Bar) menu.

Inspired the the classic menu from the PS3 - serves as visual candy in the terminal.

Why did I make xmbWavpp?

This project was actually intially a way for me to explore using a Linux distrubution for the first time.
"Ricing" is a popular concept in Linux, and many people who share their Linux modifications will often use fun visual tools like Pipes and CMatrix to take nice screenshots of their work.
This is my take on one of those classic CLI animations.

What does this project contain?

  • Pure C++ implementation (single-file)
  • Animated sine wave rendering in terminal
  • Frame-by-frame display with terminal clearing
  • Uses ANSI escape codes for cross-platform cursor movement and screen clearing
  • Adjustable terminal width detection
  • Supports smooth animation via time-based sine wave shifting
